猿题库

发表于 2013-02-26   |   分类于 Thinking   |  

猿题库自己的宣传描述语言:

“猿题库通过对近10年全部行测真题所涉156个考点的反向研究，参照项目反应理论(Item Response Theory)和等级反应模型(Graded Response Model)，综合考点考频和考题难度，利用人工智能算法，实现针对个人的能力评估和智能出题。猿题库能力评估体系将准确测算你的真实考试能力，猿题库智能出题系统能帮你提高练习效率。”

猿题库的特点

  • 真题在线模考.提供近十年所有公务员考试真题练习.
  • 快速智能练习.根据用户设置的目标考试的考点、考频、难度分布，以及之前对各个考点的掌握情况，利用人工智能算法实现的一对一智能出题，所有的题目都来源于智能出题系统对用户历史练习数据的评估分析
  • 专项智能练习.对过往公务员考试的全部题目进行细分,提供针对每个考点的针对性练习,可以自主选择要练习的题目.

流利说

发表于 2013-02-25   |   分类于 Thinking   |  

优点

  • 内容覆盖面广，而且分等级，适用于不同英语水平的用户
  • 内容不断更新，更新速度较快
  • 排行榜机制提高用户活跃度。排行榜有两个：积分榜和星星榜，积分榜表示学习数量的多少，星星榜表示学习质量的高低。
  • 可以收听其他人的朗读，增加社交互动。

  • 通关机制，在每一个分类中，必须完成上一课的挑战，才能进行下一课的学习。

豆瓣最小单元

发表于 2012-12-10   |   分类于 Thinking   |  

我一直觉得每一个社交产品都是一个独立的生态系统，如果将这个生态系统比作人体的话，那么这个社交产品中的每一个用户就相当于人体的器官，用户之间的关系就相当于人体的经脉，而社交内容，就相当于血液。这只是一个简单比喻，真实的系统远比这 个要复杂。

在每一个产品中，都有一个最小的单元，比如人体是由多种细胞构成的，脑细胞、血液细胞等等，细胞就是构成人体的最小单元。微博的最小单元是一条140字的微博，人人的最小单元是一条状态，知乎的最小单元是一个回答。这个最小单元是整个社交产品中最核心的东西，其他的所有功能都是围绕这个最小单元来构建，就像游戏中的核心玩法一样。
