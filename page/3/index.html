接受失败

发表于 2022-05-11   |   分类于 Thinking   |  

人总是很难接受一件事情失败的事实，所有人都是。

因为疫情的原因，已经在家封闭了四十多天，嘟嘟最近表现的有点烦躁不安，特征之一就是不停的要看电视，之前每天至少在户外玩耍两次，累计四个小时，现在每天憋在家里，确实也没有其他的娱乐活动了。为了保护嘟嘟的视力，限制嘟嘟看电视的时间，我们想了很多办法不让她看电视，比如拔掉电视的电源，把遥控器藏起来等等。这个时候嘟嘟就表现的很崩溃，大哭大脑，撒泼打滚，通常要持续十几分钟，吃一点蛋糕、奶棒或者喝奶才能平静下来。

正好我昨天看到「生活大爆炸」第十二季，谢尔顿因为自己的论文发表不了而对生活的一切产生质疑，他看到了自己小时候，他的爸爸做为棒球队教练如何面对失败的影片，而有了新的启发，对之前的论文有了更深的了解。他对自己的爸爸说了我印象中唯一的一次「谢谢」。

我面对失败没有更好的方法，或许一次比较正式的告别会比较有用？我不知道，下次可以试试。

有时候我表现的比较像嘟嘟，通常结果也不会改变，在哭声中不得已接受事实，这可能也是我焦虑的来源之一。

尝试换个角度看待问题吧。

怎样做「正确」的事情？

发表于 2022-03-22   |   分类于 Book   |  

首先定义一下「正确」的事情：

  1. 可以持续带来收益的高价值行动；
  2. 可以带来复利的事情；
  3. 能够积累成资产；
  4. 收益大于成本；
  5. 基于真正的需求；

产品经理的核心价值就是「做正确的事」？

在判断什么是正确的事情的时候，可以遵循以下原则：

  1. 选择不做什么，比做什么更加重要，能做减法就尽量去做减法；
  2. 远离会给你带来时间黑洞的人和事（负面效果，性价比低）；
  3. 不高估用户的品味，不低估用户的智商；
  4. 尝试理解，换位思考，不要先去评判，而是搞清楚来源、思想以及行事逻辑；
  5. 多问问为什么，找到最根源的问题（第一性原理）；

我自己不要做的事情，至少目前看来有这些：

  1. 不要反驳，不要试图说服别人，不要讲道理（嘟嘟除外）；
  2. 不免费给建议；
  3. 不说这些话：我没时间；我没办法；
  4. 不想去做英雄，知道自己的局限；
  5. 言传身教，自己没做过或者没成功的事情，不要说容易，不要去教别人，要有自知之明；
  6. 不炫耀，不自夸；
  7. 不贪小便宜；
  8. 不低估任何人，向每一个人学习；
  9. 不要分散精力在很多事情上，做好一件事情；

– 《笨方法文化手册 3.0》

今天早上突然想玩一个沙盘类的游戏，而且是要能玩很久的，满足下面的条件：

  1. 是沙盘类的游戏，或者RPG，开放性一定要好；
  2. 移动的，最好是iPad或者掌机；
  3. 能玩很久很久的；
  4. 不用那么紧张，可以放松玩，休闲类型的；
